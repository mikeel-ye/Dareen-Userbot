## VPS/Locally deploy!
```console
 $ sudo apt-get update -y
 $ sudo apt-get upgrade
 $ git clone https://github.com/mikeel-ye/Dareen-Userbot
 $ cd Dareen-Userbot
 $ python3 -m venv venv
 $ source venv/bin/act*
 $ pip3 install -U-r req*
 $ cp sample.env .env
```
