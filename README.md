## VPS/Locally deploy!
```console
 $ sudo apt-get update -y
 $ sudo apt-get upgrade
 $ git clone https://github.com/iamuput/Uputt-Pyrobot
 $ cd Uputt-Pyrobot
 $ python3 -m venv venv
 $ source venv/bin/act*
 $ pip3 install -U-r req*
 $ cp sample.env .env
```
